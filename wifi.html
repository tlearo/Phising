Secure Wi‑Fi Setup

Design a wireless configuration that keeps guest devices contained and corporate endpoints hardened.

Configuration Goals

  • Upgrade from WPA2-PSK to WPA3-Enterprise without disrupting existing trusted clients.
  • Segment guest traffic so it never touches internal networks or sensitive assets.
  • Enable monitoring hooks that flag rogue APs and anomalous client behaviour.

Artifacts Provided

You receive controller screenshots, floor plans, and a list of IoT devices that cannot join 5GHz bands. Build a configuration plan that balances user experience with security.

Document your final settings and remediation recommendations for review to bank points for this bonus mission.
